## 运维暑假考核

### Q2 邮件管理程序

> 很多时候我们都需要向用户发送邮件，但对于需要群发的邮件，手动发送总是令人恼火，所以，我们希望有一个程序来管理邮件发送

#### 基础要求

##### 1. 通过SMTP发送邮件

##### 2. 邮件编辑

##### 3. 管理收件人，批量发送邮件

##### 4. 容器化

#### 进阶要求

##### 1. 集群部署

##### 2. 友好的Web交互界面

##### 3. 添加附件、发送HTML格式邮件

##### 4. 定时发送邮件

##### 5. 提供API以方便其他程序调用

##### 6. 支持其他邮件发送协议

##### 7. ......
